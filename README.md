# 科技感跳一跳游戏

一个具有科技感视觉效果的移动端跳一跳游戏。

## 项目结构

- `index.html` - 游戏主页面
- `style.css` - 样式文件
- `script.js` - 游戏逻辑
- `particles.js` - 粒子背景效果库

## 如何运行

1. 使用现代浏览器打开 `index.html` 文件
2. 点击"开始游戏"按钮开始游戏
3. 按住屏幕或鼠标左键蓄力，松开进行跳跃
4. 尽可能跳到更多平台上获得高分

## 游戏特性

- 科技感明亮视觉风格
- 粒子背景效果
- 流畅的跳跃动画
- 响应式设计，适配移动端
- 着陆冲击波特效
- 跳跃拖尾粒子效果
